«Қара дене» — физикадағы идеал объект: ол өзіне түскен барлық сәулелерді толықтай жұтады және тек температурасына байланысты сәулелене алады. Нақты жұлдыздар осы модельге өте жақын.

I(λ,T) = (2hc²/λ⁵) · 1/(e^(hc/λkT) − 1) // Планк сәулелену формуласы (1900)

Ультракүлгін апат: Классикалық физика Рэлей-Джинс формуласымен қысқа толқындарда шексіз интенсивтілік болжады — бұл тәжірибеге қайшы келді. Макс Планк 1900 жылы энергия квантталады деген гипотеза ұсынды: E = hν.

λ_max = b / T, мұнда b = 2.898×10⁻³ м·К // Вина ығысу заңы
P = σ · T⁴, мұнда σ = 5.67×10⁻⁸ Вт/(м²·К⁴) // Стефан-Больцман заңы

Жұлдыздар түсі: Температура артқан сайын λ_max қысқарады, жарық спектрінің көк жақ шетіне жылжиды. Сондықтан ыстық жұлдыздар (10 000+ К) — көкшіл, ал салқын жұлдыздар (3000 К) — қызылдау болып көрінеді.
